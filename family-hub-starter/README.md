# Family Hub Starter

A warm, touch-friendly family life organizer starter app inspired by household dashboard products like Skylight Calendar.

## Included

- Monthly calendar dashboard
- Daily agenda panel
- Family avatars
- Chores checklist
- Weekly meal plan
- Grocery quick list
- Add-event modal
- Pastel, rounded, tablet-friendly visual style

## Run in GitHub Codespaces

```bash
npm install
npm run dev
```

Open the forwarded local port.

## Build

```bash
npm run build
```

## Suggested next features

- Save data to localStorage
- Add Firebase or Supabase for family accounts
- Add recurring events
- Add drag-and-drop calendar events
- Add family member color filtering
- Add mobile bottom navigation
- Add meal recipe links
- Add shared grocery check-off syncing
